Section What is this book about? answers natural questions such as: what is the book about, at whom it is aimed, and how to read it. Section A first model of games defines a first model of games, which is the common denominator of (almost all) models studied in this book.

We introduce the computational models that we use in Section Computational models and briefly define linear programming. In Section 1-sec:conditions we list the main objectives appearing in all chapters. A few notions will be useful throughout the book: they are developed in this chapter. We start with the notion of automata, discussed in Section Automata, and then memory for strategies, in Section Memory. We then show how automata and memory structures can be used to construct reductions between games in Section Reductions. We introduce in Section Traps and subgames the notions of subgames and traps.

The notion of fixed point algorithms is central to the study of games. We first recall the main two methods for proving the existence and computing fixed points in Section Generic fixed point algorithms. We then give an overview of two prominent families of fixed point algorithms for games: value iteration algorithms in Section Value iteration algorithms and strategy improvement algorithms in Section Strategy improvement algorithms.
